## Beschrijving

Snappthis is een webapp waar je groepen aan kan maken met thema's zodat je snapps met elkaar kan delen zodat je uiteindelijk een grootte snapmap heb.

 De website maakt gebruik van JavaScript om vette animaties te maken.
